
# Auxia GetTreatments API reference

## API Definition {api-definition}


| Header    | Value                                                                                                                                                                                                 |
|-----------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| x-api-key | A string API key that is issued to each company that is a customer of Auxia. This API key should have permission to call this API. <br/>This can be either set in request parameters or in the header |
